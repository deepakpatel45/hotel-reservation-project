# Hotel Reservation Application

## Overview

This is a command-line based hotel reservation system built using Java and OOP principles.

## Features

* Create customer account
* Search available rooms
* Book rooms
* View reservations
* Admin panel to manage rooms and customers

## Technologies Used

* Java
* OOP (Encapsulation, Inheritance, Polymorphism)
* Collections (HashMap, List)

## How to Run

1. Open project in IntelliJ
2. Run `HotelApplication.java`
3. Follow menu instructions

## Notes

* Email validation is implemented
* Date validation prevents invalid bookings
* FreeRoom is supported (price = 0)
